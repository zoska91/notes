2. CMD

- szukanie po parametrach

- parametry

musi być “$“

  1. gt - większy niż
  2. gte - większy lub równy
  3. lt - mniejszy niż
  4. lte - mniejszy lub równy
  5. in - jak w sql
  6. nin - jak w sql not in

spójniki

  1. “,” - and
  2. $or - or
  3. $not

- aktualizowanie danych

db.clients.update({x}, {y})

x - np. id elementu - czyli jakie elementy chcemy zmienić
y - co chcemy zmienić

  1. $set - aktualizuje komórkę - domyślnie tylko jedną

db.clients.update({x}, {y}, {z})

z - jest opcjonalne - ustawienia

  1. multi: true - zaktualizuj wszystkie pasujące
  1. opcjonalnie można zrobić:
  • updaneMany() - zamiast multi:true
  • updateOne()

usuwanie

db.clients.delete({x}, {y}, {z})

  • można też: deleteMany / deleteOne

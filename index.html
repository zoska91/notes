expressJs

Node - Express.js

Nasłuchiwanie serwera

Medoty req

  • req.ip - sprawdzenie mojego ip
  • req.ips - sprawdzenie ipków, przez które serwer się łączy
  • req.method - jaka metoda została wykorzystania (post/get/put itd)
  • req.url
  • req.originalUrl - ścieżka przed modyfikacją
  • req.path
  • req.protocol - czy http czy https
  • req.secure
  • req.query - pobiera parametry z linku
  • req.get - pobiera nagłówki przesłane przez klienta
  • req.params

Metody res

  • res.write - “drukuje” np. tekst na stronie, przesyła dane na front
  • res.end - “kończy połączenie - ładowanie strony
  • res.send - zamiast write i end. zmienia nagłówek “content-type”
  • res.json - przesyła tylko json
  • res.location - przekierowanie (adres http)
  • res.redirect - przekierowanie - łatwiejsze i lepsze niż location
  • res.sendFile - przesyłanie obrazków
  • res.attachment - przesłanie pliku, ale wymusi jego pobranie (sam nie zakończy połączenia. potrzebne res.end)
  • res.download
  • res.set - ustawienei dowolnych, wewnętrznych nagłóków
  • res.cookie - ciasteczka
  • res.clearCookie - usuwa ciastka (np do wylogowania)
